# Summer Dash: The Avalanche Ascent

> *Ascend the L1 ecosystem. Outpace the competition. Claim the Summit.*

## 🏔️ Project Overview
**Summer Dash** is a high-octane, endless runner web3 game built exclusively for the **Avalanche Network**. Participating in the **BuildGames** hackathon, this project demonstrates the speed, scalability, and seamless user experience of Avalanche L1s (Subnets). 

Players race through procedurally generated biomes representing the diverse Avalanche ecosystem—from the rugged **Mountains of Consensus** to the neon-lit **DeFi Distrikt**.

## 🎮 Core Gameplay
1.  **Run & Survive**: Control a "Node Runner" avatar, dashing through infinite levels. Dodge obstacles like "Network Congestion" and "Malicious Tx", while collecting **Gold Coins**.
2.  **Dynamic Biomes**: The game features environments that shift as you progress, symbolizing the journey across different Subnets and L1s.
3.  **Two Game Modes**:
    *   **Free Run**: Infinite practice mode. No wallet required to start. Connect at the end to save your collected **Gold Coins** and XP.
    *   **Summer Ranked**: Pay a small **AVAX** entry fee to compete for the daily prize pool.
4.  **Provable Fairness**: High scores and prize distributions are verified and stored on-chain.

## 🚀 Avalanche Integration & BuildGames Alignment
Developed during the **BuildGames** journey, this project validates the market opportunity for **Casual Web3 Gaming with Invisible Friction**.

*   **Sub-Second Finality**: Utilizing Avalanche's speed to ensure that entry fees and rewards are processed instantly, keeping players in the "flow state".
*   **L1 Architecture**: (Roadmap) We plan to deploy the game logic on a dedicated L1 using the **L1 Launcher**, effectively creating a "Game Chain" that handles high-throughput transactions without gas spikes.
*   **AI-Assisted Dev**: Leveraged **Avalanche AI MCP** to optimize smart contracts and generate dynamic asset parameters.

## 🛠️ Technical Stack
*   **Frontend**: React + Vite + Custom Canvas Engine (60 FPS optimized).
*   **Blockchain**: Avalanche C-Chain (Ethers.js integration).
*   **Smart Contracts**: Solidity (managing entry fees, prize pools, and immutable leaderboards).
*   **Design**: Retro-modern pixel art aesthetic tailored for browser performance.

## 🗺️ Roadmap
1.  **Hackathon Phase**: Core loop, C-Chain integration, basic leaderboard.
2.  **Phase 2**: Deployment to a custom L1 using **ICM Deployer**.
3.  **Phase 3**: Multiplayer "Race" mode using Teleporter for cross-subnet communication.
